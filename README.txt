DEPENDENCIES:
-------------   
You might need to install some of these before running the game:

  Pyglet v1.1: http://www.pyglet.org
  AVbin:       http://code.google.com/p/avbin/


RUNNING THE GAME:
-----------------
On OS X, download the Mac-specific version from http://pyweek.org/e/idg/.
On Windows, double-click "run_game.pyw".

Othewise open a terminal / console and "cd" to the game directory and run:
  python run_game.py


HOW TO PLAY THE GAME:
---------------------
Move the helicopter with the arrow keys
Fire with A, S, D & W
